# 课表页面

这是一个使用原生HTML、CSS和JavaScript开发的课表页面原型。

## 功能特点

- 显示一周七天的课表
- 点击空白处添加课程
- 点击已有课程编辑或删除
- 左上角"+"按钮可导入课程（从教务系统或扫码导入）
- 支持课程颜色自定义

## 文件结构

- `index.html`: 页面HTML结构
- `style.css`: 样式表
- `script.js`: JavaScript交互功能

## 使用方法

1. 直接打开`index.html`即可查看课表页面
2. 点击空白格子可添加新课程
3. 点击已有课程可编辑或删除
4. 点击左上角"+"按钮可显示导入选项

## 示例数据

页面加载时会自动显示一些示例课程数据，以展示页面效果。
