如果你想使用自己的Supabase项目，请按照以下步骤操作：

在使用应用之前，我们需要先为应用设置Supabase项目。

项目设置：
- 在https://supabase.com创建一个账户。
- 创建一个新项目。
- 复制schemas.sql文件的内容（你可以在项目根目录找到此文件）
- 在Supabase项目侧边栏中进入SQL编辑器
- 将复制的内容粘贴到编辑器中，然后点击运行按钮（底部会有一个绿色的运行按钮）
- 完成后，你将能够看到所有表、关系、策略、存储桶、触发器和函数。

身份验证：
- 在侧边栏进入Authentication，然后进入providers。
- 启用电子邮件提供商（如果尚未启用）
- 点击电子邮件提供商并关闭"确认电子邮件"选项，然后保存

API密钥：
- 在侧边栏进入Project Settings，然后点击侧边栏中的API选项。
- 复制Project URL并粘贴到constants/index.js文件的supabaseUrl变量中。
- 复制anon key并粘贴到constants/index.js文件的supabaseAnonKey变量中。

Supabase设置完成后，你可以运行以下命令来运行应用：
- npm install
- npm run start
